# Plugs and Sockets 🔌🌍

The definitive resource for travel plug and socket information worldwide. Check if you need an adapter before your next trip!

**Live site:** [plugsandsockets.org](https://plugsandsockets.org)

## Features

- 🔍 **Adapter Checker** - Instantly find out if you need an adapter
- 🌏 **70+ Countries** - Comprehensive coverage of popular travel destinations
- 🔌 **Plug Type Guide** - Visual reference for all plug types (A through N)
- ⚡ **Voltage Info** - Know if you need a voltage converter
- 📱 **Mobile Friendly** - Works great on any device

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No build tools or dependencies
- Static JSON data (no database)
- Deployed via GitHub Actions

## Development

```bash
# Clone the repo
git clone https://github.com/mohantyabhijit/plugsandsockets.git
cd plugsandsockets

# Open in browser (or use any local server)
open index.html
```

## Deployment

Pushing to `main` automatically deploys to production via GitHub Actions.
